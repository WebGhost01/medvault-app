# MedVault — Medical Records Manager

Secure medical records and medication tracker.

## Features
- Store and organize medical records
- Medication tracking with dosage
- Appointment management
- Health history timeline

## Stack
React 19 · TypeScript · Tailwind CSS · Vite · React Query · Shadcn/UI

## Run
```bash
npm install && npm run dev
```
Requires [portfolio-api-server](https://github.com/WebGhost01/portfolio-api-server).
